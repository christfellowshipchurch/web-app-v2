This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

## Getting Started

First, run the development server:

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Creating Components

```bash
yarn new-component [FOLDER_NAME] [COMPONENT_NAME]
# yarn new-component components ComponentName
# yarn new-component ui-kit ComponentName
```

## Directory Structure

- `__tests__` ‚Äî Top-level smoke test for rendering the `<App>`.
- `components` ‚Äî App components that are more specific than `ui-kit` components.
- `config` ‚Äî Configuration files for things like the theme, metadata, etc.
- `hooks` ‚Äî Custom Hooks for wrapping `useQuery`, `useMutation`, etc.
- `lib` ‚Äî Library code.
- `pages` ‚Äî The directory structure that dictates the routes and main screens.
- `providers` ‚Äî Components that provide data (state or GraphQL) to components.
- `public` ‚Äî Public assets.
- `scripts` ‚Äî Handy scripts.
- `ui-kit` ‚Äî Abstracted ui-kit components and library.
- `utils` ‚Äî Utility functions.
